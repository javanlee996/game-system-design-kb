## 📊 知识库结构标准

每个模块包含以下三部分：

### 1. 理论骨架
- 核心概念定义（是什么）
- 设计原则（为什么）
- 常见误区（不要怎么做）

### 2. 案例分析 ⭐
- 每个概念配1-2个具体游戏案例
- 案例格式：背景 → 设计 → 优劣分析

### 3. 设计模板
- 需求文档模板
- 设计Checklist
- 数据埋点建议
