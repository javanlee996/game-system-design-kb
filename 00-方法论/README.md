# 00-方法论：系统策划的共同语言

> **先修课**：在开始具体系统设计之前，我们需要建立一套共同的语言体系和思维框架。

---

## 一、系统策划是谁？

### 1.1 角色定位

系统策划是游戏设计师的一个分支，专注于**规则与机制的构建**。如果说数值策划负责"平衡"，关卡策划负责"体验节奏"，那么系统策划负责**"玩法的可能性空间"**。

**核心职责**：
- 设计玩家可以做什么（机制）
- 设计玩家为什么要做（动机）
- 设计玩家做完后的反馈（反馈循环）

### 1.2 与其他职能的关系

```
系统策划 ←→ 数值策划（机制需要数值平衡）
系统策划 ←→ 关卡策划（机制在关卡中体现）
系统策划 ←→ 程序（机制需要技术实现）
系统策划 ←→ 美术（机制需要视觉表达）
```

## 二、系统设计的核心思维模型

### 2.1 输入-处理-输出模型（IPO模型）

所有系统都可以抽象为：

```
玩家输入 → 系统处理 → 反馈输出
    ↑_____________________↓
           （循环）
```

**案例：星穹铁道的战斗系统**

| 环节 | 具体内容 |
|------|----------|
| **输入** | 玩家选择技能目标、释放时机 |
| **处理** | 速度条计算→技能判定→伤害计算→效果触发 |
| **输出** | 伤害数字、击破效果、角色动画、战况变化 |
| **循环** | 战斗继续，直到一方全灭 |

### 2.2 涌现性（Emergence）

简单规则的组合产生复杂、不可预测的行为。

**案例：我的世界（Minecraft）**
- 单一规则：方块可以放置和破坏
- 涌现结果：玩家建造出计算机、城市、艺术品

**反例：过度设计的系统**
- 试图预测所有玩家行为并设计对应分支
- 结果：系统臃肿，玩家感到被限制

### 2.3 反馈循环

**正反馈**（富者愈富）：
- 领先玩家获得更强资源，差距拉大
- 适用：单机成长、短周期对战
- 案例：暗黑破坏神的装备掉落，强者刷得更快

**负反馈**（平衡机制）：
- 落后玩家获得补偿，缩小差距
- 适用：竞技游戏、长线运营
- 案例：马里奥赛道的蓝龟壳（领先者被锁定）

## 三、系统设计文档规范

### 3.1 一份合格的系统文档包含什么？

```markdown
# XX系统设计文档

## 1. 设计目标
- 解决什么问题？
- 目标玩家是谁？
- 预期达成什么体验？

## 2. 核心循环
- 玩家行为流程图
- 关键决策点

## 3. 系统规则
- 详细机制说明
- 边界情况处理

## 4. 数值框架
- 关键数值关系（具体数值由数值策划填充）
- 成长曲线示意

## 5. UI/UX 需求
- 界面布局草图
- 交互流程

## 6. 数据埋点
- 需要监控的数据指标

## 7. 风险与待解决问题
```

### 3.2 好文档 vs 坏文档

| 维度 | 好文档 | 坏文档 |
|------|--------|--------|
| **目标** | 明确说明"要解决什么问题" | 堆砌功能列表 |
| **流程** | 有清晰的玩家行为流程图 | 只有静态的功能描述 |
| **边界** | 考虑了异常情况和边界 | 只考虑理想情况 |
| **可读性** | 用表格、流程图、示例说明 | 大段文字，无格式 |

## 四、常见误区（新手陷阱）

### ❌ 误区1：机制堆叠
以为系统越多游戏越好玩。

**案例**：某RPG有装备强化、装备洗练、装备镶嵌、装备附魔、装备精炼、装备觉醒...

**问题**：玩家认知负荷过重，每个系统都不深入。

**正确做法**：先做深一个核心循环，再考虑扩展。

### ❌ 误区2：忽视反馈延迟
玩家行为与反馈之间间隔太长。

**案例**：某养成系统的收益在7天后才能看到。

**问题**：玩家失去感知，无法建立"行为-反馈"连接。

**正确做法**：即时反馈 + 阶段性大反馈。

### ❌ 误区3：设计者中心
设计者觉得"这样很酷"，不考虑玩家实际行为。

**案例**：某PVP系统设计了复杂的属性克制关系，但玩家实战中无法感知。

**问题**：设计沦为背景设定，不参与实际决策。

**正确做法**：从玩家视角验证，机制必须可感知、可理解、可利用。

## 五、Checklist：系统上线前的自查

- [ ] 玩家能否在30秒内理解这个系统的基本规则？
- [ ] 系统是否有明确的开始和结束？
- [ ] 玩家是否清楚自己的进度和状态？
- [ ] 是否存在明显的最优解（导致其他选择失去意义）？
- [ ] 系统是否与游戏的其他系统产生有意义的交互？
- [ ] 如果移除此系统，游戏体验会受到什么影响？
