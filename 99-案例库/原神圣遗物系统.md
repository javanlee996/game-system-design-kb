# 案例：原神的圣遗物系统

> **涉及模块**：02-成长系统、03-经济系统（间接）、06-商业化设计

---

## 1. 背景

- **游戏类型**：开放世界ARPG
- **开发商**：米哈游
- **上线时间**：2020年9月
- **商业表现**：全球移动端收入超40亿美元

**设计目标**：
- 提供长期的角色成长目标
- 延长游戏寿命，防止"毕业即弃坑"
- 创造可付费的商业化空间（体力/树脂）

---

## 2. 系统拆解

### 2.1 核心循环

```
刷取圣遗物（消耗体力）→ 
查看词条（主词条+副词条）→ 
决定是否保留 → 
强化升级（消耗摩拉和其他圣遗物）→ 
追求完美副词条 → 
重复刷取（无尽循环）
```

### 2.2 关键机制详解

#### 机制A：套装效果

- 2件套：基础属性加成
- 4件套：机制性改变（改变玩法）

**案例**：绝缘之旗印4件套
- 基于元素充能效率提升大招伤害
- **设计意图**：改变角色的配装逻辑，让"充能"从生存属性变为输出属性

#### 机制B：词条随机系统

| 词条类型 | 说明 | 设计意图 |
|----------|------|----------|
| **主词条** | 圣遗物类型决定可选范围 | 控制大方向，避免完全随机 |
| **副词条** | 4个，随机生成，强化时随机提升 | 创造"彩票"体验 |
| **初始词条数** | 3-4个随机 | 已强化的圣遗物有"沉没成本" |

#### 机制C：强化随机性

- 每4级提升一次副词条
- 从已有词条中**随机选择**一个提升
- **关键设计**：新词条只会在0级时出现，强化只提升已有词条

**数学期望**：
- 双暴（暴击率+暴击伤害）圣遗物是顶级追求
- 获得一个"能用"的双暴圣遗物需要刷取数周
- 获得"完美"双暴圣遗物可能需要数月甚至以年计

### 2.3 体力系统（树脂）

- 每天自然回复180点树脂
- 刷取一次圣遗物本消耗20点树脂
- **设计意图**：限制刷取速度，延长内容消耗周期

---

## 3. 心理机制分析

### 3.1 可变奖励机制（斯金纳箱）

圣遗物系统是游戏设计中**可变比率强化**的经典案例：

```
行为（刷本）→ 不确定的奖励（随机词条）→ 
多巴胺分泌 → 重复行为
```

**为什么比固定奖励更上瘾？**
- 下一次可能就是"极品"的不确定感
- 近失效应（出了好主词条但副词条差，"差一点就中了"）

### 3.2 沉没成本陷阱

- 玩家会对已强化的圣遗物产生依恋
- 即使获得更好的，也因为"已经投入这么多"而不舍得更迭
- 延长每次迭代的周期

### 3.3 目标梯度效应

- 从"主词条对"→"双暴初始"→"双暴强化命中"，每个阶段都是新目标
- 玩家永远不会"彻底毕业"，总有优化空间

---

## 4. 优劣评估

### ✅ 做得好的地方

1. **延长游戏寿命**
   - 提供了几乎无限的成长目标
   - 免费内容的可持续性极强

2. **付费转化点**
   - 体力购买是核心付费点
   - 角色越多，圣遗物需求越大，付费意愿越强

3. **社区话题性**
   - "晒圣遗物"是社区核心内容
   - 玩家间有强烈的比较心理

4. **策略深度**
   - 不同角色需要不同套装
   - 词条的边际效应计算有深度

### ❌ 存在的问题

1. **挫败感极强**
   - 数周零提升是常态
   - "防御之王"等梗反映玩家痛苦

2. **新玩家追赶困难**
   - 与老玩家的圣遗物差距几乎无法追赶
   - 入坑门槛越来越高

3. **玩法单一化**
   - 双暴成为几乎所有输出角色的唯一追求
   - 其他词条（防御、生命、精通）的价值被边缘化

4. **时间绑架**
   - 玩家感到"不刷就亏了"
   - 形成上班感

---

## 5. 可迁移经验

### 可以借鉴的设计

| 经验 | 适用场景 |
|------|----------|
| **分层随机** | 主词条保底+副词条彩票，平衡确定性和惊喜 |
| **套装改变机制** | 不只是数值，让装备改变玩法 |
| **边际收益递减** | 永远有优化空间，但越往后越难 |

### 需谨慎借鉴的设计

| 设计 | 风险 |
|------|------|
| **纯随机词条** | 挫败感强，可能导致玩家流失 |
| **体力限制** | 容易被视为逼氪，需要平衡 |
| **无保底机制** | 极端情况下玩家可能永远毕不了业 |

---

## 6. 改进思路（批判性思考）

如果重新设计圣遗物系统，可以考虑：

1. **词条保底机制**
   - 刷取一定次数后，可以定向选择一个词条
   - 保留随机性，但消除极端非酋

2. **词条转移/洗练**
   - 允许消耗资源转移或重置副词条
   - 让玩家对随机性有更多掌控

3. **多样化毕业标准**
   - 不只是双暴，让其他词条组合也有竞争力
   - 不同流派有不同最优解
